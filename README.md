# Applied-Data-Science-Lab
## WorldQuant University – Data Science Portfolio (Projects 1 to 7)

This repository contains all projects completed as part of WorldQuant University's Applied Data Science Lab programme.  
Each project follows a professional structure with dedicated data folders, notebooks, scripts, and outputs.

## List of completed projects

1. **Housing in Mexico** – Data cleaning, EDA, pricing insights  
2. **Predicting Housing Prices in Buenos Aires** – Feature engineering, regression models  
3. **Air Quality in Nairobi** – Time series, MongoDB, AR – ARMA  
4. **SQL & Risk Modelling** – SQLite, Logistic Regression, Decision Trees  
5. **Predicting Earthquake Damage in Nepal** – ML classification, model tuning  
6. **Customer Segmentation in the US** – K-means, PCA, Dash App  
7. **DS Lab Applicants Experiment & A/B Testing** – ETL OOP, chi-square tests, choropleth maps, dashboards, MongoDB

## Structure of each project
Each project contains:
```bash
project-name/
├── data/ # Raw or cleaned data
├── notebooks/ # Analysis notebooks
├── src/ # Python scripts
├── outputs/ # Images, reports, saved models
└── README.md # Documentation

```
